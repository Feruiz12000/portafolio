Ivoting

Es un sitio web en donde un ciudadano puede registrarse con su codigo único de identificación (CUI) y crear un nuevo usuario, puede ver la información de cada elección y la información de cada candidato y realizar su voto garantizándole la privacidad del mismo.



Pagina de inicio

En la página de inicio, el ciudadano podrá iniciar sesión o registrar su usuario. El sistema buscará la información del ciudadano en la base de datos del RENAP y la comparará la fotografía que se acaba de tomar con la fotografía almacenada en la base de datos.



Vista de elecciones

El usuario ciudadano podrá ver todas las elecciones que estén habilitadas o deshabilitadas y toda la información de cada elección.



Resultado de elecciones

El usuario podrá ver el resultado de cada elección por medio de 2 tipos de graficas (grafica de barras y grafica de sectores).



Realizar voto

El usuario podrá realizar un único voto por elección, y por motivos de seguridad debe de brindar nuevamente su correo y contraseña para confirmar su voto.



Ciclo DevOps

Para el proyecto se ejecutó el ciclo DevOps con el fin de monitorear cada parte del proyecto, ejecutando las pruebas unitarias de cada micro-servicio, luego se realizó un análisis del código, utilizando Sonarqube y se construyeron los artefactos de cada micro-servicio.


Se utilizó un clúster de kubernetes para publicar la página web, y un balanceador de carga para cada servicio, y la estrategia de Rolling update.



Prometheus y Grafana

Para monitorear el tráfico y la cantidad de peticiones a la base de datos se utilizó Prometheus y grafana, ya que estas herramientas brindan un entorno grafico fácil de entender.
